# Tripleten web_project_homeland

## Descripción

Proyecto de Sprint 5 que consiste en una pagina web responsiva que se ajusta a distintos tamaños de pantalla y despliega informacion de ciudades natales de miembros de TRIPLETEN.

## Funcionalidades

- Diseño responsivo para dispositivos móviles, tablets y escritorio
- Navegación fluida entre secciones

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 (Flexbox)
- Metodología BEM
- Diseño responsivo con media queries
